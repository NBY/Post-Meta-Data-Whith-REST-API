== Description ==

This plugin is used to add meta data (postmeta) when publishing posts via REST API.

Python3 REST API posts meta data example code please visit:

https://github.com/NBY/Post-Meta-Data-Whith-REST-API

== Installation ==

1. Upload post-meta-data-whith-rest-api to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
